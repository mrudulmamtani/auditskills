# AuditSkills Platform — Quick Start

## Prerequisites
- Node.js 18+ installed

## Setup & Run
```bash
cd auditskills-app
npm install
node server.js
```

Then open: **http://localhost:3000**

## What you get
- **Dashboard** — live stats: skills, runs, findings, exposure
- **Skill Registry** — browse/filter/promote audit skills
- **Skill Builder** — 4-step wizard to create/edit skills
- **Audit Runs** — upload Excel/CSV data or use synthetic demo data
- **Findings Register** — review, filter, update status, add management responses
- **Excel Working Paper** — download per run (multi-tab: population, tests, register)
- **PDF Findings Report** — download professional findings report per run

## Quick Demo
1. Click **▶ New Run** in Audit Runs
2. Select "Vendor Invoice Completeness" skill
3. Leave file blank (uses synthetic demo data)
4. Click Start — watch it run in ~2 seconds
5. Download Excel or PDF outputs
